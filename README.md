# Neon Snake Rush

A polished browser-based snake game with:

- keyboard and mobile touch controls
- music and sound effect toggles
- high score persistence
- Reddit score sharing
- a neon mode and a real snake mode
- an AI snake skin prompt and upload flow

## Run locally

Open `index.html` in any modern browser.

## Publish with GitHub Pages

1. Open the repository settings.
2. Go to **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and the `/ (root)` folder.
5. Save.

Your site should then publish at:

`https://mathorline.github.io/neon-snake-rush/`
